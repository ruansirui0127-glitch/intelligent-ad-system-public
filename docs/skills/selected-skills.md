# Selected Project Skills

These skills are installed locally for the intelligent advertising system under `skills/`.

## Installed Skills

- `frontend-app-builder`: use for new dashboards, local visual tools, product surfaces, and major UI rebuilds.
- `frontend-testing-debugging`: use for rendered frontend verification, local dev server checks, responsive QA, console errors, and UI regressions.
- `vercel-react-best-practices`: use when writing, reviewing, or refactoring React/Next.js code, especially performance-sensitive dashboard code.
- `project-learnings`: use to record local corrections, recurring failures, missing capabilities, and reusable workflow lessons in `.learnings/`.
- `api-contract-designer`: use for API contracts, mock payloads, shared wire shapes, Meta adapter output, and frontend/backend handoff validation.

## Project Fit

These are intentionally project-local because this repository has a specific operating context: intelligent advertising workflows, Meta-first integration, later platform expansion, diagnosis/optimization logic, and local/private configuration boundaries.

Do not store real tokens, real ad data, or private account details in skill files or `.learnings/`.
