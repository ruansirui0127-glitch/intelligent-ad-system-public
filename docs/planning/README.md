# 规划材料说明

`docs/planning/` 在公开 main 中只保留安全说明和脱敏模板。

原始 PRD、业务画像、竞品清单、账户口径、导出报表、截图、Word/PDF/Excel/ZIP 等材料不进入公开仓库。需要协作时，放在团队私有文档系统或私有目录中，再把可公开的结论沉淀到 `docs/product-vision.md`、`docs/architecture.md`、`docs/data-model.md` 和 `docs/api-contracts.md`。

公开文档可以写“系统需要支持什么能力”，不要写“真实账户目前是什么数据、真实客户是谁、真实预算和阈值是多少”。
