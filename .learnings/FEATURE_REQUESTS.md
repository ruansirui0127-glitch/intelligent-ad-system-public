# Feature Requests

Requested capabilities, workflow gaps, and future improvements for the intelligent advertising system.

Use this file when:
- the user asks for a capability that does not exist yet
- a workflow would benefit from a new script, skill, automation, dashboard, connector, or template
- an idea should enter the backlog without interrupting the current task
