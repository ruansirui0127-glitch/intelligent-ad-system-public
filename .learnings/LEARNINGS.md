# Learnings

Corrections, insights, knowledge gaps, and reusable best practices captured while building the intelligent advertising system.

Use this file when:
- the user corrects scope, platform priority, KPI interpretation, wording, or workflow assumptions
- a better reusable approach is discovered for Meta Ads, Xiaohongshu, cross-platform data, diagnosis, budget, or creative workflows
- a project convention should be remembered locally before it is promoted to long-term memory or a reusable skill

## Entry Template

```markdown
## LRN-YYYYMMDD-001 - category

**Logged**: ISO-8601 timestamp
**Priority**: low | medium | high | critical
**Status**: pending | applied | promoted
**Area**: meta | xiaohongshu | data | diagnosis | optimization | frontend | workflow | infra

### Summary
One-line description of the learning.

### Context
What happened, what was wrong or missing, and what should be done next time.

### Suggested Action
Specific change to a skill, script, prompt, checklist, data model, or project convention.

### Metadata
- Source: conversation | error | user_feedback | review
- Related Files:
- Tags:
```
